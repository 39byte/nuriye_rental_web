# 누리예 카메라 대여 시스템 (Nuriye Camera Rental System)

## 1. 프로젝트 개요
본 프로젝트는 한국외국어대학교 카메라 동아리 '누리예'의 장비 대여 프로세스를 자동화하고 효율적으로 관리하기 위해 제작된 웹 애플리케이션입니다. 부원들은 실시간으로 장비 예약 현황을 확인하고 신청할 수 있으며, 집행부는 관리자 대시보드를 통해 대여 전 과정을 체계적으로 관리할 수 있습니다.

## 2. 주요 기능
### A. 부원용 기능 (대여 신청 및 현황)
- **실시간 대여 캘린더**: 월간 단위로 장비 대여 현황을 시각화하여 제공합니다.
- **스마트 예약 시스템**:
  - 카메라 바디와 렌즈의 호환성을 자동으로 체크하여 선택 가능한 목록을 필터링합니다.
  - 대여 희망 기간에 장비가 이미 예약되어 있는지 중복 여부를 실시간으로 검사합니다.
  - 각종 액세서리(SD카드, 리더기, 삼각대 등) 선택 기능을 포함합니다.
- **테마 지원**: 라이트 모드, 다크 모드 및 시스템 설정 연동 테마를 지원합니다.

### B. 집행부용 기능 (관리 대시보드)
- **신청 관리**: 승인 대기 중인 신청 건을 확인하고, 담당자 지정 및 승인/반려 처리를 수행합니다.
- **진행 중 대여 관리**: 현재 대여 중인 장비의 반납 예정일까지 남은 기간(D-Day)을 확인하고 반납 완료 처리를 합니다.
- **자산(인벤토리) 관리**: 동아리 소유 장비의 목록을 실시간으로 편집하고 업데이트할 수 있습니다.
- **보안 설정**: 관리자 전용 비밀번호를 통해 접근을 제어하며, 비밀번호 변경 기능을 제공합니다.

## 3. 기술 스택
- **Language**: Python 3.x
- **Web Framework**: Streamlit
- **Database**: Supabase (PostgreSQL)
- **Data Analysis**: Pandas
- **UI/UX**: Custom CSS, Streamlit Fragments

## 4. 설치 및 실행 방법
### 사전 요구 사항
- Python이 설치되어 있어야 합니다.
- Supabase 계정 및 프로젝트가 필요합니다.

### 실행 순서
1. **의존성 라이브러리 설치**:
   ```bash
   pip install -r requirements.txt
   ```
2. **환경 변수 설정**:
   `.streamlit/secrets.toml` 파일을 생성하고 아래 형식을 입력합니다.
   ```toml
   [connections.supabase]
   SUPABASE_URL = "YOUR_SUPABASE_URL"
   SUPABASE_KEY = "YOUR_SUPABASE_ANON_KEY"
   ```
3. **애플리케이션 실행**:
   ```bash
   streamlit run app.py
   ```

## 5. 관리자 정보
- **초기 비밀번호**: `1111` (로그인 후 설정 탭에서 변경 권장)
- **데이터베이스 구조**: `Inventory`, `Rentals`, `Settings` 테이블이 Supabase에 구성되어 있어야 합니다.
